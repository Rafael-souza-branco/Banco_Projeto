#Banco virtual

Um simulador de banco virtual onde o usuário faz um mini cadastro e escolhe entre sacar, depositar (baseado no valor da conta) e sair

##Funcionalidades
- Cadastro rápido (nome, idade, Gênero, senha e saldo)
- Da 3 opções de entrada as repetindo até que o usuário deseja sair
- Não saca valor maior que o saldo da conta
- Após sair do programa exibe o saldo atualizado antes de fechar

##Como usar
1. Execute o arquivo 'Banco_interativo.py'
2. Faça o cadastro
3. Escolha entre (Saldo, Sacar ou Sair)

##Tecnologias usadas
- Python 3

##Objetivo
Treinar loops, entrada de usuário e classes
